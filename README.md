# FinanceTracker 2026

A production-ready, no-login personal finance tracker that runs directly on GitHub Pages.

## Features

- Offline-first PWA with install support and service worker caching
- IndexedDB autosave with local session recovery
- Mobile responsive dark and light UI
- Dashboard for income, expenses, balance, savings, budget, cashflow and net worth
- Transactions with add, edit, delete, search, sort, filter, categories, tags, notes and attachment names
- Charts: pie, doughnut, bar, line and area
- Monthly analytics and report panel
- Budget planner, EMI tracker, recurring bill reminders and local notifications
- Undo delete, drag and drop import and keyboard shortcuts
- Export to CSV, Excel, JSON and PDF
- Import from CSV, Excel and JSON

## Keyboard Shortcuts

- `Ctrl + N`: focus quick transaction entry
- `Ctrl + S`: save current session
- `Esc`: clear the transaction form

All user data is stored locally in the browser. There is no backend, no account system and no tracking.
